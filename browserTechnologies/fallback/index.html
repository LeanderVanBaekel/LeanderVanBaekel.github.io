De fallback is:

CSS features

viewport with & height

De viewport units worden niet in elke browser ondersteunt. De fallback hier voor kan zijn:

.image {
	width: 100%; width 100vw;
	height: 200px; height 30vh;
}

Op deze manier kijkt de browser alleen naar de VW en VH als hij er ook iets mee kan doen. Anders valt hij terug op de eerder genoemde regel.

Can I Use viewport-units? Data on support for the viewport-units feature across the major browsers from caniuse.com.

Javascript features

classList

Niet elke browser ondersteunt de classList feature in javascript. Als dit het geval is stopt de browser volledig met het uitvoeren van je script.

Om dit te voorkomen kun je:

document.getElementById.className += " .."

Can I Use classlist? Data on support for the classlist feature across the major browsers from caniuse.com.

GPS

omdat javascript een enhancement is voor de website mag het bijna niet voorkomen dat er een functie echt een fallback nodig heeft. Progressive enhancement zou wel kunnen.

Voorbeeld:

De gebruiker kan op een website zijn locatie invullen, en de server geeft vervolgens informatie terug. Als progressive enhancement zou javascript de locatie van de gebruiker via gps kunnen bepalen.
