Github

Githubbers gebruiken Github

Een van de developers verteld over hoe hij zijn werk doet bij Github. hij legt uit dat ze bij het ontwikkelen van Github.com zelf ook Github gebruiken.

Hij verteld over het belang van het gebruiken van branches om bij te houden hoe een nieuwe feature wordt ontwikkeld.

Hij verteld hoe zijn vrouw, die bij reddit.com werkt, nieuwe features ontwikkeld. Ze werken daar niet met Git maar heel erg liniar en brengen eens in de zoveel tijd een hele nieuwe versie van de site uit waar alle updates in verwerkt zitten.
