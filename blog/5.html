PPK (Peter Paul Koch)

Web vs. Native

Volgens PPK zijn de browsermakers nu vooral bezig met het toevoegen van functies aan hun browsers waarmee ze meer kunnen lijken op native apps. PPK vind dit een slechte manier van denken omdat je hiermee de voordelen van het web uit het oog verliest.

Hij vind ook dat de websites te zwaar zijn geworden. Hij denkt dat dit door ads komt maar ook doordat we veel te veel tools gebruiken.

javascript

PPK vind dat een echte javascript programmeur alleen frameworks en libraries gebruikt als het echt nodig is. Ook kan hij zelf delen uit libraries halen om zo min mogelijk overbodige code toe te voegen.

PPK vind de kracht van het web: URL's, bereik en eenvoud.
